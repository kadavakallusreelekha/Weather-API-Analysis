# Weather-API-Analysis
A data analytics project that uses a Weather API as a data source and visualizes real-time weather data entirely in Power BI. The project includes data transformation using Power Query, calculated measures using DAX, and interactive dashboards to analyze city-wise and day-wise temperature trends and weather conditions.
